Steps to run the application

Install Homebrew. 

Follow the steps on the https://treehouse.github.io/installation-guides/mac/homebrew to install Homebrew.

Download Node:

    $ brew update
    $ brew install node
    $ node -v
    $ npm -v

Download Expo.io:

    $ npm install --global expo-cli
    $ cd into the project 
    $ expo start -c
